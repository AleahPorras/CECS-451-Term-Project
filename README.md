# How to Run

**Website Link:** https://ai-flashcardgenerator.streamlit.app/
1. Upload pdf file of notes/lecture slides/textbook chapter.
2. Select how many flashcards you want to generate (1 minimum, 10 maximum).
3. Review generated flashcards.
4. Choose which questions you struggled with under **Office Hour Questions**. Click Generate Office Hour Questions.
5. Review generated questions.

1. requirments.txt
   * This is the file that has all the packages we used that you have to install first 
3. flashcard.py
   * Our main program
   * Directions and tips will be printed at the top of the terminal to understand how to run and what to input
   * Error statements are also in place to help guide the user
   * Purpose: Generates flashcards based on a pdf, then based on what the user struggled with it also generates some questions to ask during office hours based on those topics
